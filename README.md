# AI Career Recommendation System

## Overview

This project is a simple AI-inspired recommendation system built using Python.

The system analyzes a user's skills and interests, compares them with predefined career profiles, calculates similarity scores, and recommends the most suitable career paths.

---

## Features

* User skill and interest analysis
* Career profile matching
* Similarity score calculation
* Ranked career recommendations
* Personalized results
* Pure Python implementation

---

## Technologies Used

* Python
* Functions
* Lists
* Dictionaries
* Sorting Algorithms
* Score-Based Recommendation Logic
